Sangkhan

เป็นเกมแนว Action Adventure Fantasy ที่ได้รับแรงบันดาลใจจากวรรณคดีไทย ผู้เล่นจะได้รับบทเป็นดวงวิญญาณลึกลับทีสามารถ 
“สิงร่าง” สิ่งมีชีวิตในตำนาน เพื่อใช้พลังและความสามารถเฉพาะตัวในการต่อสู้ สำรวจ และแก้ไขปริศนาภายในดินแดนโบราณ 
โดยที่ผู้เล่นจะได้ควบคุมสิ่งมีชีวิตในตำนานที่จะมีสกิลที่สามารถใช้ได้ และจะสามารถรับสกิลใหม่ๆออกมาใช้ได้เรื่อยๆ จากการสิงร่างเมื่อพบเจอสิ่งมีชีวิตในตำนาน ผู้เล่นจะต้องใช้ไหวพริบ และสมาธิ เมื่อเจออุปสรรคที่ไม่คาดคิดอย่างไม่ทันตั้งตัว รวมกับการวางแผนตัดสินใจในการสิงร่าง สิ่งมีชีวิตในตำนานให้เหมาะสมกับเหตุการณ์ต่างๆ

ปุ่มควบคุม (Controls)
W, A, S, D / ปุ่มลูกศร: เคลื่อนที่ตัวละคร
Spacebar: กระโดด
คลิกซ้าย (L-Click): โจมตีธรรมดา
คลิกซ้าย (R-Click): โจมตีหนัก

ฟีเจอร์เด่นของเกม (Key Features)
ระบบต่อสู้แบบ Combo: การโจมตีต่อเนื่องที่ลื่นไหลพร้อมเอฟเฟกต์ตระการตา
ระบบสิงสู่: ผู้เล่นจะได้ความสามารถในการสิงสิ่งมีชีวิตอื่นเพื่อใช้ความสามารถที่หลากหลาย
เสียงประกอบเฉพาะตัว: เพลงประกอบและเสียง Effect ที่ช่วยเพิ่มความตื่นเต้นระหว่างเล่น
เทคโนโลยีและเครื่องมือที่ใช้ (Tech Stack & Tools)

Game Engine: Unity 6.0

Programming Language: C# / JavaScript

Graphics/Art Tools: Blender (3D Models)

Audio: Audacity, Reaper (SFX & Music)

🚀 วิธีการติดตั้งเพื่อพัฒนา หรือดาวน์โหลดไปเล่น (Getting Started)

OS: Windows 10/11 (64-bit) หรือ macOS


ขั้นตอนการรันโปรเจกต์ใน Editor (For Developers)

เปิด Scene หลักที่ตั้งอยู่ในโฟลเดอร์ Assets/Scenes/Main.unity แล้วกดปุ่ม Play ใน Editor ได้ทันที
